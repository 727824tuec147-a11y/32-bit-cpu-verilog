# 32-bit CPU Design in Verilog

Designed and simulated a custom 32-bit processor using Verilog HDL.

## Features
- FSM-based CPU (Fetch–Decode–Execute)
- Custom instruction set
- Register file and ALU
- Memory-mapped UART output
- 256x32-bit RAM
